\# Smart Library System



\## About Project



This is a simple console-based library system built using Java.

I created this project to understand how real-world systems like a library can be implemented using Object-Oriented Programming.



The system allows basic operations like searching books, issuing and returning them, and managing users.



\---



\## What this project can do



\* Login system for different users (Borrower, Clerk, Librarian)

\* Search books by title, author or subject

\* Issue and return books

\* Place hold requests if a book is not available

\* Calculate fine for late returns

\* Add and update borrower details

\* Add and manage books in the library



\---



\## Technologies used



\* Java (Core Java)

\* OOP concepts (Inheritance, Encapsulation)

\* ArrayList for storing data

\* JDBC (for database connection)



\---



\## How it works (basic idea)



The program starts from Main.java and shows a menu in the console.



After login:



\* Borrower can search and view books

\* Clerk can issue/return books

\* Librarian can manage books and system data



All data is handled through different classes like Library, Book, Borrower, and Loan.



\---



\## How to run



1\. Open the project in any Java IDE

2\. Make sure Java is installed

3\. Run the Main.java file

4\. Follow the options shown in the console
